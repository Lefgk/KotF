## Installation

To Test smart contract in Hardhat :

- Run `npm i` and `npx hardhat test --network hardhat`
  inside `/testing` <br/>

To Try the live version in BSC testnet in the front -end :<br/>

- Run `npm i` and `npm start` inside `/Front-End`
